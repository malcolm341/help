          Gridify UVs


          UV maps selected faces into a uniform grid of UVs.

          Select some faces in the 3d viewport or the UV editor and run the tool to gridify UVs.

          Existing UVs are not required for this tool to work so you can gridify UVs on unmapped faces if you want.

          Limitations:

          -Doesn't work on triangles.
          -Doesn't work on n-gons.

          If the newly created grid of UV's is facing the wrong way, click the gridUV button a second time to rotate it by 90 degrees.


          This tool uses Maya's auto seam feature to automatically cut a UV seem when needed. For example, on cylindrical shaped meshes that don't have a natural UV border edge.

          If you're not happy with the resulting auto seam you can manually choose which edge loop to use as the UV seam:
          -Select your mesh, hold right mouse button and choose Multi.
          -Select the edge loop you want to use as the UV cutting seam.
          -Hold shift and select the faces you want to gridify so you have edges and faces selected at the same time.
          -Selection order doesn't matter if you prefer to select faces first.
          -Run the tool.


          The texel density the UVs are created with is 5.12 pixels per unit, if you'd like to modify this value locally:
          -Right click the gridUV button on the shelf and choose edit.
          -A window opens displaying the code for the tool.
          -Near the top of the code look for float $pixPerUnit = 5.1200;
          -Change 5.1200; to whatever texel density you prefer.
          -Next time you run the tool it will use the updated value you set.


          If you'd like to keep the existing texel density of the selected faces and apply that to the grid of UVs:
          -Right click the gridUV button on the shelf and choose edit.
          -A window opens displaying the code for the tool.
          -Near the top of the code look for $gridUVs_keepExistingShellTD = 0;
          -Change the 0 to a 1.
          -Next time you run the tool it will use the size of the selected faces in the UV editor as the texel density. If no UVs exist for the selected faces it will normalize the resulting grid of UVs to fill the 0-1 UV space.
