          About

          The Mel Script Mega Pack is a collection of scripts and tools for Autodesk Maya.
          I created the Mega Pack to make Maya faster and easier to use, but it's also fun to automate tasks that would otherwise be time consuming and tedious inside of Maya. All the tools are written in MEL script which means they work on newly released versions of Maya without needing to be updated.

          Installation

          Compatability:
          -Works on the newest version of Maya and old versions of Maya back to 2019
          -Works on Maya LT and Student
          -Works on Maya 2018 update 6 and Maya 2017 update 5
          -Works on PC and Mac

          How to Install the Scripts:
          Close Maya.

          Download the .zip file you purchased to the your desktop.

          Open the .zip and copy the shelf file contained in it to your desktop.

          Open Maya.

          Click the gear icon in the top left corner of Maya and choose load shelf.

          Browse to the shelf file you just copied to your desktop and load it.

          You should now see a new shelf in Maya with the same name as the file on your desktop, it contains the scripts you purchased.

          If you don't see any shelf names click the gear icon and turn on shelf tabs.

          If you want to copy my scripts to a different shelf, hold Ctrl + Shift and Middle Mouse Drag any of my shelf buttons to any other shelf. Make sure to keep holding Ctrl + Shift when you release the middle mouse button or the scripts will be moved rather than copied.

          How to Update the Scripts to the Latest Version:
          If you previously purchased a script pack you can update to the latest version at any time for free. Updates include bug fixes, new features, and even new scripts if you purchased the Mega Pack, Modeling Pack, or UV Pack.

          Delete the existing shelf you want to update in Maya first.

          Close Maya.

          Download the .zip file you purchased to your desktop. Use the original e-mail link you received, or look for the file in your Gumroad or Artstation Library and download it from there.

          Open the .zip and copy the shelf file contained in it to your desktop.

          Open Maya.

          Click the gear icon in the top left corner of Maya and choose load shelf.

          Browse to the shelf file you just copied to your desktop and load it.

          You should now see a new shelf in Maya with the same name as the file on your desktop, it contains the latest versions of the scripts you purchased.

          If you don't see any shelf names click the gear icon and turn on shelf tabs.

          Be aware that any scripts that you've moved or copied to other shelves are not updated. Unfortunately you'll have to delete those buttons manually and copy my scripts back to your shelves again each time you want to update. Alternatively, if you only have a couple buttons you want to update you can right click and choose edit on any of the buttons on my shelf and copy paste the updated code directly into your shelf buttons.


          Quick Start Guide

          Help Button

          Takes you to the online help page.

          Marmoset Export Toolbox

          Speeds up exporting meshes for baking in Marmoset Toolbag or Substance Painter including name matching, material ids, and normal tools inside Maya.

          UV Edges Toolbox

          Harden normals on UV border edges while softening normals based on angle tolerance setting you choose.

          Normal Map Viewport Mode

          Applies normal map lighting to selected objects, from there you can render or screen capture the top viewport to generate normal map stamps and textures.

          Obj Import Export Toolbox

          Speeds up exporting meshes as .obj for use with applications like Zbrush. Export each mesh as a separate file, triangulate, import multiple, custom file location, and more.

          One Click Export to C:\drive

          Exports selection to a temp.ma, temp.obj, or temp.fbx file on your hard drive. Handy for moving objects between scenes or saving a temp file without any file browsing.

          One Click Import from C:\drive

          Imports the temp.ma, temp.obj, or temp.fbx file the above button creates without any file browsing.

          Xray Selected Objects

          Toggle Xray mode on/off for selected objects. A bit better than the default Xray, this script will invert the selected objects Xray and if nothing’s selected it will Xray everything.

          Wireframe Selected Objects

          Toggle wireframe mode on/off for selected objects. Handy for looking through selected objects without changing any viewport settings.

          Reference Planes Toolbox

          Create multiple mesh reference planes each with a different image at the correct aspect ratio in one click. Great for rotoscoping, choose axis, scale width, and more.

          Delete Empty Groups

          Deletes empty groups, transforms, and ghost nodes that are often left behind even after running Maya’s optimize scene and delete history.

          Fast Open Maya

          A quick way to disable the plugins that significantly slow Maya down when it's opening, especially if you’re using a Maya version prior to 2022.

          Fast Close Maya

          Closing Maya prior to Maya 2022 was extremely slow. The fast close tool can close Maya instantly in one click for Maya 2020, 2019, 2018, 2017.

          Advanced Save Preferences

          Saves contents of script editor, MTK custom shelf, recent file history, tool settings, and other preferences that aren’t included in Maya's default save preference command.

          Increase Shelf Height Toolbox

          Increase or decrease shelf height at any time to fit height of current shelf in one click. Custom shelf height is retained after Maya restart unlike other tools I've seen online.

          Extra HUD Toolbox

          New HUD elements for working units, scene time warp on/off, up axis, selection counter, assigned materials, current UV set, preserve UVs on/off, project location, time since last save, time since last auto save, tool axis, camera distance, and camera based selection on/off.

          Dynamic Measuring Toolbox

          Creates a measurement node between two or more verts with a single click. Measurements update automatically so you can move meshes and verts around after creation.

          Move Pivot to Selection Center

          Moves pivot to selection center in one click, works on components or objects.

          Rebuild Rotation Axis from Component

          Rebuilds rotation axis along selected component, works even if the transform is frozen. Fixes channel box values unlike the Maya default align to component tool.

          Rotate Mesh to Face Y Axis

          Rotates the whole mesh so the selected component faces Y up, even if the transform is frozen. This is handy for un-rotating a mesh so you can work on it aligned with the world axis.

          Snap Objects to Surface Normal

          Snaps multiple objects to the selected surface aligned to the normal, similar to the snap together tool, but works on multiple objects and is faster to use.

          Duplicate Object to Components

          Places a copy of your object at each selected component's center aligned to the normal.

          Vert Snapping Toolbox

          Snaps selected vertices to closest verts on a separate mesh. Saves lot of time if you’re working with two separate meshes that need to be seamless.

          Weld Selected to Closest

          Welds selected components to closest vertices without collapsing to a single point. Handy for quickly optimizing a mesh or a whole edge loop.

          Weld Selected to Target

          Welds selected components to target vertex. Easier and faster to use than the Maya default target weld because you don’t have to drag select the target.

          Create Primitive at Selection Toolbox

          Create a primitive at the selected component of another mesh aligned to its normal. Choose pivot location and scale. Handy for creating primitives aligned to face normals.

          Axis Aligned Lattice Deformer

          Create a lattice deformer aligned to selected edge, vert, or face. Works on object or components, also works even if the transform is frozen.

          Mirror and Instance Toolbox

          Quickly mirror objects and speed up symmetrical modeling, with or without instancing. Faster to use than the Maya default mirror menu and has extra useful features.

          Clean Combine

          Combines meshes without creating a bunch of temp nodes in the outliner. Keeps your scene clean, preserves object name, pivot location, and scene hierarchy.

          Clean Extract and Separate

          Extracts selected faces without creating a bunch of temp nodes in the outliner. In object mode this tool separates polygon elements into separate meshes.

          Clean Duplicate Faces

          Duplicates selected faces in one click without creating a bunch of temp nodes in the outliner.

          Select Every Other Edge Vert Face Toolbox

          Selects alternating patterns of components, for example 1 on 1 off, or 3 on 3 off, or 4 on 4 off. Selects loops or rings.

          Select Half Your Mesh from Edge Loop

          Converts any continuous edge loop selection to half of your mesh in face mode.

          Show Geometry Border Edges

          Identify unwelded verts, failed mirroring operations, and other problem areas on your mesh. Easier and faster to see than the Maya default way.

          Naming Toolbox

          Rename multiple objects with increment in one click. Easier and faster to use than Maya’s default tool. Search and replace part of a name, add a prefix or suffix, and more.

          Copy Material from Face to Clipboard

          Copy a material from one face to the clipboard, allows for copy/pasting materials in the viewport without opening the hypershade.

          Paste Material from Clipboard

          Paste a material from the clipboard to faces or objects. Save time by applying materials in the viewport without opening the hypershade.

          Replace Material id from Clipboard

          Replace a material id on a whole object from the clipboard. Save time by applying materials in the viewport without opening the hypershade.

          Change Default Material Toolbox

          Set the default material with a single click, or save a custom material and load it into any scene as the default. Useful if you like to model with a specific material applied.

          Select Faces by Material id

          Selects all faces that share the same mat id on a single mesh, so you can work with the selection even if the same mat id is found on other meshes in the scene.

          Graph Materials from Selected Faces

          Graphs materials in the hypershade from faces. Faster than Maya’s default tool because it only graphs what you have selected in the viewport.

          Checker Map Toolbox

          Toggle a custom checker map image on/off for your objects in one click, or choose from four built in checker styles. Remembers your original material assignments, tile, rotate, and more.

          UV Grabber Toolbox

          Grabs UV snapshots with one click and sends them directly to Photoshop or the image viewer of your choice. Saves a lot of time and works with UDIM tiles.

          Udim Packing Toolbox

          Pack UDIM tiles interactively while maintaining texel density, or pack tiles automatically based on world space location. Significantly faster and easier to use than Maya’s default tool.

          Light Mapping Toolbox

          Create lightmap UVs in one click. Use the angle method for minimal seams, or start from map1 and automatically fix areas that aren't mapped properly for baking.

          Fast and Easy UVs Toolbox

          Automatically create UVs for your mesh, then cut and sew semi-interactively to make adjustments. Useful for organic shapes or if you prefer faster UV creation with a bit less control.

          Gridify UVs

          UV maps faces into a seamless grid of symmetrical squares. Works on any face selection, but may require unfolding and straightening after.

          Rectangularize UVs

          UV maps faces into a seamless grid with correct aspect ratio. A bit harder to use than gridUV, but doesn’t require any unfolding or straightening after.

          UV Mapping Toolbox

          Custom and existing UV tools organized for quick access. Create UVs faster than Maya’s UV Toolkit and use the custom tools to accomplish what Maya can’t.

          Reset
          Resets the toolbox back to defualt settings. If Maya crashes and some of the checkboxes become inverted, use the reset button to fix this issue.

          Working Units  ✔ 
          Turn on if you want to see the current working units Maya is set to. For example, meters, centimeters, feet, etc.

          Scene Time Warp  ✔ 
          Turn on if you want to see if the enable scene time warp feature is on or off for the current scene.

          Up Axis  ✔ 
          Turn on if you want to see the current up axis Maya is set to. For example, y or z.

          Selected Items  ✔ 
          Turn on if you want to see the number of currently selected items, anything you select will be automatically counted and the number displayed.

          Assigned Materials  ✔ 
          Turn on if you want to see the names of the materials that are assigned to the currently selected objects or faces. The material names displayed will update as your selection changes.

          UV Set  ✔ 
          Turn on if you want to see what UV set is currently active for the selected mesh.

          Preserve UVs  ✔ 
          Turn on if you want to see if the preserve UVs feature is on or off.

          Project  ✔ 
          Turn on if you want to see the current project path Maya is set to.

          Last Save  ✔ 
          Turn on if you want to see how long it's been since you last saved the scene. The timer will reset if you save, save as, or increment save.

          Turn on if you want to see how long it's been since Maya last auto saved the scene. If auto save is disabled in the preferences the timer will display auto save disabled. If you're using a Maya version older than Maya 2019 this feature won't work and the timer will display requires Maya 2019 or later.

          Cam Distance  ✔

          Select Every Other Edge Vert Face Toolbox


          Compatability:
          -N-gons are not supported and will be ignored during pattern selection
          -Triangles are not supported, if you have any triangles selected while using the tool the selection will come out wrong
          -Non-symmetrical patterns are not supported, so 4/on 3/off style patterns are not possible in this tool

          invert dot/pat
          Inverts the last pattern that was selected by the tool.

          dot loop
          Selects every other component on the current loop. The current loop is defined by selecting two adjacent components of the same type and running the tool.

          pat loop
          Selects a user defined pattern of loops on the current mesh. The loop pattern is defined by selecting two or more adjacent components of the same type and running the tool. For example, selecting 4 adjacent faces will produce a 4/on 4/off pattern. Non-symmetrical patterns are not supported, so 4/on 3/off is not possible in this tool.

          to loop
          Converts the current selection to a component loop of that type, this can be helpful to convert Non-symmetrical or partial selection patterns into component loops.

          to ring
          Converts the current selection to a component ring of that type, this can be helpful to convert Non-symmetrical or partial selection patterns into component rings.

          dot ring
          Selects every other component on the current ring. The current ring is defined by selecting two adjacent components of the same type and running the tool.

          pat ring
          Selects a user defined pattern of rings on the current mesh. The ring pattern is defined by selecting two or more adjacent components of the same type and running the tool. For example, selecting 4 adjacent faces will produce a 4/on 4/off pattern. Non-symmetrical patterns are not supported, so 4/on 3/off is not possible in this tool.
